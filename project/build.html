Step-by-Step Build Guide

Step 1: Prepare Your Workspace

Find a clean, static-free surface. Gather all your tools and parts before starting.

Step 2: Install CPU & RAM

Gently install the CPU and RAM into the motherboard before placing it into the case.

Step 3: Mount Motherboard

Secure the motherboard into the case using the included standoffs and screws.

Step 4: Install Storage

Install your SSD or HDD. Connect SATA cables if using a SATA-based drive.

Step 5: Install Power Supply

Secure the PSU into the case and ensure the switch is set to off.

Step 6: Install GPU

Place your graphics card into the PCIe slot and secure it with screws.

Step 7: Connect Cables

Connect all power cables, case connectors, and data cables as per the motherboard manual.

Step 8: Perform POST Test

Turn the system on and make sure all fans spin and monitor displays POST screen.

Step 9: Install Operating System

Use a USB or disc to install Windows or your preferred OS.

Troubleshooting Tips

Double-check All Power Connections

Ensure your 8-pin CPU and 24-pin motherboard power cables are properly connected.

No Display on First Boot?

Make sure your monitor is plugged into the GPU, not the motherboard port.

RAM Not Detected?

Try reseating the RAM and using the correct slots (usually 2 and 4).

PC Not Turning On?

Double-check your PSU switch and front-panel connectors.

Stay Grounded

Prevent static damage by touching grounded metal before handling components.

BIOS Update

Sometimes required for newer CPUs on older motherboards — check the manufacturer’s site.

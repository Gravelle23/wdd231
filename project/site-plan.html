Beginner’s Guide to Building a PC – Site Plan

Site Name

Name: BeginnerPCBuild.com

This name clearly communicates the site's purpose: helping beginners understand how to build a personal computer. It is easy to remember and would work well as a brand.

Site Purpose

This site provides simple, step-by-step tutorials for building a PC, including guides for selecting compatible parts, assembling hardware, installing software, and troubleshooting. It aims to help first-time builders gain confidence and avoid common mistakes.

Scenarios

  • What components do I need to build a beginner gaming PC?
  • How do I install Windows after assembling my PC?

Color Schema

  • Primary Color: #1e2a38 (Dark Navy) – used for headers and footer background
  • Accent Color: #4caf50 (Green) – used for buttons, links, and highlights

Typography

  • Heading Font: Roboto Slab – used for all main headings
  • Body Font: Open Sans – used for general body text and paragraph content
